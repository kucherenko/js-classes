## Lesson 3
[Presentation](http://slides.com/andreykucherenko/deck-31-32)

### Installation

```
cd lesson-3
npm install
```
### Build script
```
npm run build
```
### Run http server

```
npm run server
```

### Build code for browser

```
npm run browserify
```
Check `lib/` folder for js after `npm run build`, `npm run browserify` and `npm run server`
